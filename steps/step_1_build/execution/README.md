# Этап 1. Сборка docker-образа — самостоятельная работа.

**Исходные данные:** исходный код фронтенд приложения в директории `frontend`.

**Цель:** создать Dockerfile, который позволит упаковать простое Node.js-приложение в Docker-контейнер. Такой контейнер можно будет запустить на любом сервере, не устанавливая Node.js вручную.

**Задачи:**

1. Создайте файл `Dockerfile` в директории frontend
2. Заполните `Dockerfile`:  
   2.1. Базовым образом должен быть лёгкий вариант Node.js версии 18 `node:18-slim`.  
   2.2. Задайте рабочую директорию внутри контейнера `/app`.  
   2.3. Скопируйте файлы зависимостей package.json и package-lock.json (если есть) в контейнер.  
   2.4. Контейнер должен выполнить установку всех пакетов, необходимых вашему приложению. Установить зависимости через `npm install`.  
   2.5. После установки зависимостей скопируйте весь код приложения в контейнер.  
   2.6. Запустите приложение. Команда запуска — выполнение файла server.js с помощью Node.js, `node server.js`.
3. Соберите образ
4. Запустите контейнер, пробросив порт `3000` наружу.
